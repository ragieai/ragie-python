                     | Partition strategy for the document.<br/>Different strategies exist for textual, audio and video file types and you can set the strategy you want for <br/>each file type, or just for textual types.<br/><br/>For textual documents the options are `'hi_res'` or `'fast'`.<br/>When set to `'hi_res'`, images and tables will be extracted from the document.<br/>`'fast'` will only extract text.<br/>`'fast'` may be up to 20x faster than `'hi_res'`.<br/>`hi_res` is only applicable for Word documents, PDFs, Images, and PowerPoints.<br/>Images will always be processed in `hi_res`.<br/>If `hi_res` is set for an unsupported document type, it will be processed and billed in `fast` mode.<br/><br/>For audio files, the options are true or false. True if you want to process audio, false otherwise.        <br/><br/>For video files, the options are `'audio_only'`, `'video_only'`, `'audio_video'`.<br/>`'audio_only'` will extract just the audio part of the video.<br/>`'video_only'` will similarly just extract the video part, ignoring audio.<br/>`'audio_video'` will extract both audio and video.<br/><br/>To process all media types at the highest quality, use `'all'`.<br/><br/>When you specify audio or video stategies, the format must be a JSON object. In this case,<br/>textual documents are denoted by the key "static".
